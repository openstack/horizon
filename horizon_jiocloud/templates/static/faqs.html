General

  • Who can use Jio Cloud?

    Jio Cloud is a platform for developers who wish to launch and scale their applications quickly and with minimal expenditure. Jio Cloud is the ideal platform to bring start-up ideas to life with zero capital expenditure.

  • Why should I use Jio Cloud?

    Jio Cloud is India's biggest public cloud and has all its data centres in India. This provides very low latency to your applications. Additionally, Jio Cloud offers the cheapest rates in the industry, while at the same time offering the best SLA's.

  • Do your offer DNS as a Service?

    Jio Cloud does not offer a DNS as a service at the moment.

  • What kind of virtualization technologies does Jio Cloud use?

    Jio Cloud uses OpenStack as the underlying software for the IaaS, and KVM as the hypervisor.

  • How do I get started with Jio Cloud?

    You can sign up for an account on the website. You will immediately be able to login to the dashboard and create instances, volumes and objects. For further help on how to make use of these features for your application, refer to the Jio Cloud Starter Guide.

  • Why should I verify my phone number while signing up?

    Jio Cloud would like to have a valid phone number and e-mail address for every customer, in case they need to be contacted. You will not receive any promotional calls or messages on your phone. The verification is extremely simple. All you need to do is enter a code sent to your number via text message.

  • What can I do with Jio Cloud instances?

    Please refer to our starter guide to get a brief introduction to how you can make use of Jio Cloud instances for a variety of use cases.

  • What should I do if I forget my Jio Cloud account credentials?

    Please send a mail to jiocloud.support@ril.com and a new password will be generated and sent to your registered e-mail address.

Technical

  • How do I take backups?

    Jio Cloud allows you to take snapshots of your instances and volumes at any time you wish. Once a snapshot is taken, it is stored in the Object Store where three replicas are stored to ensure a backup copy exists in case a copy is lost.

  • How do I make use of APIs?

    Please refer to the API documentation guide to understand how APIs are used, what the input parameters are and what the expected response codes are.

  • Will I have console access to my instance?

    Yes, Jio Cloud provides you console access to your instances through the Dashboard.

  • What is the difference between local storage and block storage?

    Local storage physically resides on the same server as your virtual machine. It is ephemeral storage and will be wiped upon termination of your virtual machine. Whereas, Jio Cloud Block Store is a persistent storage and your data in it will continue to exist even after your virtual machine is terminated, and can be used later with new virtual machines.

  • How long does it take to launch an instance?

    It may take up to 50 seconds to launch a new instance.

  • How quickly can I scale my capacity up and down?

    While Jio Cloud allows you to scale your capacity up and down, the speed depends on various factors like size of the server, time of the day, data and software to be installed on scaling up, etc. Scaling down can take longer than scaling up because the system needs to consolidate and copy data to a smaller size, clean up unused old logs/sessions files to help resize timings, etc. The peak times are start and end of any business day.

  • What OS environments does Jio Cloud support?

    Jio Cloud currently supports Ubuntu, Windows and CentOS. You can contact Jio Cloud support if you wish to use any other OS.

  • How many public IP addresses can I use?

    You are currently limited to 10 public IP addresses. You can contact Jio Cloud support if you wish to use more public IPs.

  • What happens to my data when a system terminates?

    When your instance is terminated, all data in the local storage will be wiped. However, the data in your Block Store and Object Store will be available for use at a later time.

  • Can I attach multiple instances to a single volume?

    Jio Cloud currently does not allow you to attach multiple instances to a single volume.

  • Can I use my existing Windows Server license with Jio Cloud?

    Absolutely, Jio Cloud allows you to use your own Windows license.

  • Can I use my existing RHEL license with Jio Cloud?

    Absolutely, Jio Cloud allows you to use your own RHEL license.

  • What if my Jio Cloud instance is malfunctioning/not responding?

    If you ever face this situation, please contact us at jiocloud.support@ril.com and our excellent support team will sort out the issue as quickly as possible.

  • Can I re-size my instance?

    Jio Cloud does not allow you to re-size your instance. However, Jio Cloud makes it very easy to scale horizontally.

  • Is there any limitation on single file transfer size?

    Jio Cloud does not add any limit on the size of a file for transfer.

  • Is FTP allowed to Jio Cloud?

    Jio Cloud provides you complete access to your instances. Hence, FTP access is allowed as well.

  • Does Jio Cloud support live migration?

    Jio Cloud does not allow users to live migrate their instances from one physical server to another. However, the Jio Cloud admins have the ability to do that for maintenance of physical servers.

Security

  • How do I prevent other people from viewing my systems?

    By default, all your Jio Cloud instances and storage are visible only to you. However, you can make your files stored in the Jio Cloud Object Store public and allow others access to it.

  • Does Jio Cloud offer encryption on Block Storage?

    Unless the client encrypts data prior to transmission, Jio Cloud does not perform any encryption before storing it. Users must select and perform the type and level of encryption best suited for their application.

  • What level of support is provided by Jio Cloud?

    Jio Cloud provides a 24x7 support for any query you might have. Jio Cloud guarantees a response within three hours of logging the request.

  • Can I protect my instances with firewalls?

    Jio Cloud enables you to create security groups which will determine what ports need to be active and what ports need to be closed, allowing you to add a layer of security to your applications.

  • Can Jio Cloud lose my data?

    Jio Cloud will guarantee data durability as per committed SLAs.

  • How does Jio Cloud secure my data?

    Jio Cloud doesn't encrypt user data but it takes various measures to ensure your data is kept safe. All traffic between clients and Jio Cloud systems use SSL to establish a secure, encrypted communication channel. The Jio Cloud Identity service ensures that users authenticate with their login credentials (username/password) and access keys to generate a session authentication token. These authentication tokens are used to validate all operations on Jio Cloud.

  • What should I do if I lose my private key?

    Jio Cloud does not store your private keys. If you forget your paraphrase, you cannot restore your data. Please contact Jio Cloud support to help you reset your keys.
